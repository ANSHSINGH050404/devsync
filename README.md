# devsync

DevSync — developer collaboration and project sync platform.

## Stack

Next.js, Drizzle, TypeScript

## Getting started

```bash
git clone https://github.com/ANSHSINGH050404/devsync.git
cd devsync
npm install
npm run dev   # or: npm start
```
